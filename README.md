# Maquette -> Frontend

Page HTML + Bootstrap reproduisant la maquette fournie.

Aperçu rapide:

- Ouvrir `index.html` dans un navigateur moderne.
- Utilise Bootstrap 5 via CDN et un CSS personnalisé `css/style.css`.

Pour un serveur local rapide (optionnel):

```powershell
# depuis le dossier projet
python -m http.server 8000
# puis ouvrir http://localhost:8000
```
